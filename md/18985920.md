# 国企科研岗被主管恶意绩效考核打低分如何破局

Thread ID: 18985920

## 1楼 - pikaSB[离线] (2025-12-22 10:41:24)

背景：科室主管打分占比总分20%，可以理解为满分100分的20分。科研成果打分占比70分，另有10分基础分。大家都是做科研项目，科研成果分数差异很小，且会有归一化操作，确保最高分就是70分，所以大家差异极小，一般也就是2分不到。那么，科室主管的20分就影响极大。通常，各科室这一部分都是按照几乎满分来给，例如20分、19.9分等，以求不在这里拉开差距。
现状：科室主管恶意给科室被针对的成员打19分和18分，看着和20分似乎差异不大，但是别的科室基本可以认为是满分，导致科室平均分就比其他科室低1-2分，可以让科研成果打分的第一名直接变成最后一名。
如果打个15分，10分，很明显，还好闹一下，但是19、18，看起来似乎还觉得打的高了，但是其实已经是天堑差距。
