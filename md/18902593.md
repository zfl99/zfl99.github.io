# 从招人面试看出国家这些年发展太快了

Thread ID: 18902593

## 1楼 - Crayon[离线] (2025-03-08 08:56:39)

在美国招人，new grad或者recent grad的入门岗位，十年前中国学生和印巴伊朗之类的学生差不多，都是问无不知，经验很多，各种知识概念没做过也听说过，都是十八般武艺高强的全能选手；现在招人，即使是清华北大的，态度都有了很大变化，比较laid back, 不知道就是不知道，自然就说出来了，对职位也不是很看重，问面试官问题也很多是和生活质量相关。说实话吊儿郎当劲和美国白人有些接近了，比较潇洒，高高兴兴的无所谓，不少基础知识也没听说过。印巴之类的学生和以前一样，没做过的也要强行装出来很有经验的样子，问面试官一些明显是网上那些教程里公式化的问题来表现自己渴望这个职位，吹捧面试公司有多强大。
和一些在学术界的朋友老师谈也是同感，中国学生不如以前好压榨了，很多天坑专业大家都不去了，想找牛马剥削要靠印巴孟加拉了。和教授顶嘴，撂挑子不干的人也越来越多。
还真是那句，少年强则国家强，士别三日，当刮目相看。国家经济发展了，青年人的心态也更正常化了。
