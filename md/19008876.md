# 关于教培行业的一些感悟分享

Thread ID: 19008876

## 1楼 - willing[离线] (2026-04-21 09:48:53)

本人目前负责一家小小的教培工作室的运营，最近看到有很多人讨论教培，所以想分享一些关于教培的常见问题，参照前面老哥的声明：以下内容源于个人工作见解，带有主观性和局限性，有不准确、不适合的地方还请大家多多批评指正。
一、教培公司给的高薪保底，是不是真的可以拿到
教培的薪资大部分按照底薪+课时费的模式，你可以直接问招聘的人事，会给你每周安排多少小时的课程，每小时课时费的标准，进行简单乘法运算就可以了。注意以下的点：
（一）课时费是否跟班级人数挂钩、是否跟学生续费挂钩，或者是否还有其他考核标准；课时费的涨幅如何评定，多长时间评定一次（学而思、新东方一般是一个季度考核一次）
（二）寒暑假的课量是多少，一般寒暑假课量会是平时每月的1.5-3倍，寒暑假大量上课，如何安排休息时间。（一般是开课前和结课后调休，如果说没有休息，这家不要去）
（三）是否配有班主任或者课程咨询顾问等角色，帮忙减轻授课老师服务家长或者续报的压力。（续费就是催着学生家长交钱，这是个苦差事，即使多给钱，如果这个活要自己干，还是谨慎考虑）
（四）底薪需要做哪些工作，比如教研、招生讲座，是否要参与（也就是除去上课时间，其他时间怎么安排）。
（五）一些机构会跟你谈课时费分成，说按照每节课班课收费的比例给你；好处，确实有机会多拿钱，坏处，其实是把班主任、课程招生顾问等角色提供的一些服务转移到教师身上完成，所以会非常辛苦。（如果把教培当成快速捞钱的地方，那这种模式挺好的）
二、教培的晋升空间在哪里
教培行业内：转管理，晋升成为校长；努力上课，变成高课时费的名师；
跳转学校：很多私立校其实非常欢迎教培行业的教师，尤其是清北学历的，不过需要老师之前有带出过比较好的成绩；
自主创业：大部分的教培名师，都会在某个阶段从原来的机构跳出来，自营；
经历有限，暂时还看不出其他的晋升空间，欢迎大家补充互动。
三、其他行业和教培行业，怎么抉择？
以下内容全是个人参考建议，本人自毕业以来完全没有在教培以外的行业工作过，对其他行业的了解全部来自浅显的观察法，谨慎参考。
（一）如果你另一份offer是比较具有技术壁垒的工作，比如医生，那么不太建议入行教培；教培就像985的外卖行业，只要你学历够硬，在原来的行业不开心，都可以选择教培，入行门槛极低，竞争极大。（具体问题还得具体分析）
（二）如果你另一份offer有比较直观可控的晋升路径（尤其是晋升有明显的社会价值提升），那么不太建议入行教培；（或许你能在更高的平台为社会创造更多价值）
（三）如果你热爱教育工作，喜欢教书，喜欢育人；你不喜欢学校给你很多限制，如果你想给你的学生提供更多自由的思想，更多有趣的内容；那么你应该挺适合教培的。
四、教培行业能不能自己创业入行
可以的。起步时你只需要两个角色：教师、运营。这两个角色可以是同一个人，也可以是两个人或者多个人。教师上课，运营拓客。如果是自己一个人，会比较辛苦，但是全流程收入自己拿，稳定下来在一线城市年入百万不难。如果运营和教师是两个人或者多个人，那么至少在起步阶段，得配合非常默契，互相理解对方的工作要求（我和我的搭档配合默契，因为很早就是情侣了，现在是夫妻）。教师这个角色一定要很强，强到学生有免费的班课都不去上，更愿意上你的课（好多大机构也有名师，还给优秀的学生提供免学费政策）。关于如何成为一名很强的教培教师，可以参考赛艇先生的公众号文章。
