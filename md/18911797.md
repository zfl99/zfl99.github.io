# 程序员的出路在哪里？

Thread ID: 18911797

## 1楼 - FLYFLY[离线] (最后修改于2025-04-04 20:45:082025-04-04 15:22:10)

贵校本科毕业后已经在某互联网公司工作接近3年，最近有些呆不下去想要跑路。背景是小镇做题家，父母无养老压力但也无法提供更多帮助。
现公司缺点：
入职来技术上感觉没有很大的进步，也没有升职，因为公司的奇葩政策近年来升职普遍缓慢，据说第一次升职一般至少要3年，后续也不容易。不谈能力的话，级别上的进步显著慢于其他同类公司。而且即使在公司一直呆着混level也不见得能有工资上的显著提升，涨薪显然主要靠跳槽，感觉到了该离开的时候。
现公司优点：
每天6:30下班几乎不加班，摸鱼时间长，年假比较多，工资水平还算能接受。
跳槽主要考虑的问题：
工作强度：
国内互联网行业虽然需求量比较大，但加班非常严重，有些难以接受日常10点下班。同等工作强度的岗位非常难找，很可能短期内找不到。海外机会在2025年每个地方都已经极其困难，直接通过工签run出去的可能性很低，而且除了美国之外收入一般都比国内低，美国也不可能去得了。
个人成长性：
在1的基础上，即使接受国内的工作强度，在跳槽后可能也同样得不到成长，工作大多数时候一样是重复劳动。相反，现在的工作之余还有更多个人时间，反而为成长提供了更多可能性，但也只是可能性而已。如果在现在的公司停留太久，将来被裁员后可能因为能力/level差于在卷厂的同事，想加班也找不到地方收留。
职业发展前景：
对于自己这种非行业top的普通程序员，想不出来在2025年的今天应该持有怎样的职业预期。互联网行业高速发展的红利已经过去，公务员一类的稳定行业个人无法接受体制内的氛围，run出国难度极大，独立开发难在发现需求和市场营销而不是技术，接外包的性价比过低。说起来好像是既要又要了，但其实自己也不知道自己想要的到底是什么。或许程序员最好的归宿就是在职业生涯结束前加班+节衣缩食攒够足够退休的钱，职业生涯结束后再另谋出路？
