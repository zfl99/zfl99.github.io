本科

区间：6,800–33,000 元（极少低于 7,000）
硕士
平均：1.7 万–1.8 万元 / 月
区间：8,700–54,000 元
热门（金融 / 计算机）：3 万–4.5 万 / 月
博士
平均：2.6 万–2.7 万元 / 月
科研 / 高校：1.5 万–3 万
企业研发 / 算法：2 万–4 万 +
三、分专业大致起薪（2025 届，本科 / 硕士）
计算机 / AI / 电子
本科：25–30 万 / 年（≈2.1–2.5 万 / 月）
硕士：35–45 万 / 年（≈2.9–3.8 万 / 月）
金融 / 经济
本科：20–25 万 / 年（≈1.7–2.1 万 / 月）
硕士（顶级投行 / 基金）：60–80 万 / 年
工科 / 建筑
本科：18–22 万 / 年
硕士：28–35 万 / 年
文科 / 社科 / 法学
本科：12–18 万 / 年（≈1.0–1.5 万 / 月）
硕士：20–25 万 / 年
四、关键特点
学历溢价明显：博士 > 硕士 > 本科，差距显著
下限很高：本科几乎无低于 7,000 元，硕士极少低于 1 万
专业差距极大：计算机 / 金融 ≫ 普通文理科
深造率高：本科约 80%+ 读研 / 出国，直接就业人数偏少
